How It Works

Three recommendation algorithms powering NexMovie

📐

Content-Based Filtering

TF-IDF + Cosine Similarity

Builds a feature vector for each movie using genres (3× weight), director (2× weight), cast, description, and decade. Computes cosine similarity between vectors to find structurally similar films.

1Extract text features per movie
2TF-IDF vectorize all features
3Cosine similarity against target
4Rank by similarity score
👥

Collaborative Filtering

User-Based Cosine Similarity

Builds a user-movie rating matrix. When you rate movies, your rating vector is compared to existing users via cosine similarity. Movies highly rated by similar users are weighted and recommended.

1Build user × movie rating matrix
2Compare your ratings to all users
3Weighted average of similar users
4Recommend unseen high-score films
🔀

Hybrid Recommendation

Content (60%) + Collaborative (40%)

Blends both approaches for better accuracy. Content-based scores weighted at 60% ensure genre/style alignment, while collaborative scores at 40% ensure social proof from users with similar taste.

1Run content-based (top 12)
2Run collaborative (top 12)
3Blend: 0.6 × CB + 0.4 × CF
4Re-rank by blended score
